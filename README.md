# 📈 Construção de Carteiras de Investimentos com NotebookLM

## Objetivo

Utilizar o NotebookLM para estudar investimentos e construir carteiras diversificadas para diferentes perfis de risco.

## Fontes

- Tesouro Direto
- B3
- CVM
- O Investidor Inteligente - Benjamin Graham
- A Psicologia Financeira - Morgan Housel

## Estrutura do Projeto

- 📄 Relatório Final
- 📚 Glossário
- 💼 Carteira Conservadora
- ⚖️ Carteira Moderada
- 🚀 Carteira Arrojada
- 🤖 Prompts utilizados no NotebookLM

## Principais Aprendizados

- Diversificação
- Relação risco-retorno
- Rebalanceamento
- Importância do horizonte de investimento
- Perfil do investidor

## Tecnologias

- NotebookLM
- Markdown
- GitHub

  ##Prompts
1-)Explique os principais tipos de investimentos no Brasil e suas características.
2-)Atue como um planejador financeiro. Explique renda fixa, ações, FIIs e ETFs, apresentando risco, liquidez e horizonte de investimento em uma tabela.
3-)Considere um investidor moderado com horizonte de 20 anos. Como uma carteira diversificada poderia ser estruturada?
4-)Com base nas fontes carregadas, crie um resumo dos principais conceitos de investimentos para um iniciante.
5-)Crie um glossário contendo os principais termos encontrados nas fontes, como CDI, Selic, ETF, FII, Dividend Yield e P/L.
6-)Monte uma carteira para um perfil conservador/moderado/ Agressivo e explique os motivos da alocação.
Utilize todas as respostas geradas anteriormente nesta conversa, incluindo as carteiras para os perfis conservador, moderado e arrojado, e incorpore-as ao relatório final.

Reescreva o conteúdo em formato de documento estruturado contendo:

Introdução
Principais Classes de Investimentos
Conceitos Fundamentais
Carteira Conservadora
Objetivo
Alocação em tabela
Justificativas
Riscos
Carteira Moderada
Objetivo
Alocação em tabela
Justificativas
Riscos
Carteira Arrojada
Objetivo
Alocação em tabela
Justificativas
Riscos
Gerenciamento de Riscos e Rebalanceamento
Conclusão

Apresente o resultado em formato Markdown, pronto para exportação ou publicação no GitHub.

Com base em todas as fontes e na nota 'Carteiras de Investimento', gere um relatório final completo em Markdown.
